## PAYable CLI

* Logs
```shell
$ payable logs
```

* Scrcpy
```shell
$ payable scrcpy
```

* APK
```shell
$ payable apk some_file.apk
```

* Files
```shell
$ payable files any_file_name
```

* Install from URL
```shell
$ payable install-url "https://someurl.com/file.apk"
```

* Rename file according to the output.json version name and custom name
```shell
$ payable vname
```
